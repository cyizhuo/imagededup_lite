# imagededup_lite

## Introduction

Simple fuction for finding near/exact duplicated images

- Modified from <https://github.com/idealo/imagededup>
  - `imagededup` is powerful but too slow, and support only python
- With both `python` and `c++` versions
  - c++ version might differents a little bit from python version
  - due to OpenCV and Pillow difference
- With both `chineses` and `english` comments

## 介绍

用于查找近似/完全重复图片的简单功能函数

- 修改自 <https://github.com/idealo/imagededup>
  - imagededup 功能强大但速度太慢，并且只支持 Python
- 同时提供 `python` 和 `c++` 版本
  - 由于 OpenCV 和 Pillow 的差异，C++ 版本运行结果与 Python 版本稍有不同
- 同时提供中英文注释版本
